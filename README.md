# linked-scraper
A Python web scraper for LinkedIn. A working one.

The web scraper uses Python3.8 and selenium to achieve its task. It scrapes your search query and stores links to the found people in a txt file.
<br/> This file might also be used to connect your account with found persons.
<br/> Use **python linked-scraper.py -h** to get an overview of possible options.
